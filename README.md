# 📊 HR Analytics Dashboard – Power BI

This project demonstrates an end-to-end **data analytics workflow**, starting from **data cleaning and preprocessing in Python** to **interactive visualization in Microsoft Power BI**.

Instead of directly building visuals, the dataset was first cleaned, validated, and transformed using a Jupyter Notebook — mirroring real-world analytics practices.

---

## 🚀 Project Workflow

The project follows a practical analytics pipeline:

1️⃣ **Raw HR Dataset**  
2️⃣ **Data Cleaning & Preprocessing (Python / Jupyter Notebook)**  
3️⃣ **Feature Engineering & Validation**  
4️⃣ **Power BI Data Modeling & DAX Measures**  
5️⃣ **Interactive HR Analytics Dashboard**

---

## 🧹 Data Cleaning & Preparation

Before building the dashboard, the dataset was processed using:

📄 `HR_project_Bi.ipynb`

Key preprocessing steps included:

✅ Handling missing / unknown values  
✅ Data type corrections  
✅ Removing inconsistencies  
✅ Creating derived fields  
✅ Preparing analytics-ready dataset  

This ensures the dashboard is built on **reliable and structured data**, similar to production analytics environments.

## 📊 Dashboard Overview

The Power BI dashboard provides a comprehensive analysis of employee data, focusing on:

- Employee distribution
- Attrition patterns
- Compensation insights
- Demographic trends
- Department & job role analysis

- **Total Employees:** 1398

## 📈 Key Insights

✔ Highest attrition observed in **26–35 age group**  
✔ Lower salary bands show greater attrition risk  
✔ Sales & R&D departments contribute most to attrition  
✔ Single employees show higher attrition trend
